// for c23 standard. apparently the most widely used version of c is c99, so
// beware when going to older hardware and embedded systems that only support
// c99.
gcc -Wall -Wextra -std=c23 -pedantic foo.c

// also note that ubuntu still comes with gcc-13 by default. install gcc-14 via
sudo apt install gcc-14 g++14
// and then compile with
gcc-14 -Wall -Wextra -std=c23 -pedantic foo.c

// if using math.h, you need to link with the math library on unix like systems
// because the complier doesn't automatically do so. just add -lm flag. it
// sounds like the reason is to avoid linking a bunch of floating point code if
// it's not necessary. and i guess this means that the compiler doesn't
// automatically link whatever you #inlcude.
gcc-14 -Wall -Wextra -std=c23 -pedantic foo.c -lm
